HOW TO ADD YOUR PAINTINGS  —  IvosArt
======================================

Save each painting into THIS "images" folder using the EXACT filename below
(all lowercase, .jpg). The website loads them automatically — no code editing
needed. Until a file exists, the site shows an elegant placeholder, so nothing
ever looks broken.

Tip: for the best look, crop each photo to JUST the artwork (you can leave the
frame in — it looks great). Aim for ~1200px on the long edge.

WANT TO CHANGE TITLES, SIZES OR PRICES?
Open index.html, find the "WORKS DATA" list near the bottom, and edit the
title / medium / year / price text. To mark a piece as sold, add  sold:true
to its line, e.g.:
   {img:"work-03.jpg", title:"Crimson Ascent", ... , sold:true},

That's it — refresh the page and your gallery is live.
